Skeleton
=========
 


## 依赖 
```html
<link href="stylesheet" type="text/css" href="main.css">
<script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript" src="skeleton.js"></script>
```
      

## Skeleton.ripple
为组件添加水波纹效果 

```html
<button class="btn">Button</button> 
``` 
```javascript 
Skeleton.ripple({
    elem:".btn"//渲染的渲染元素     
})    
```
